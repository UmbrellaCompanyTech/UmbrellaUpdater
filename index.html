How to use

1. Click the "Select Firmware" button and choose the firmware.
2. Unplug the power cable.
3. Connect the pedal to your PC using a USB-C cable.
4. While holding down both footswitch, power on the pedal.
5. ※Windows only Install Windows driver Windows only. Not necessary on Mac.
  1. 5.1. Connect the pedal to your PC in bootload mode (perform steps 1-4 of this procedure).
  2. 5.2. Download and open Zadig
  3. 5.3. Click "Options"→"List All Devices"
  4. 5.4. Select "DFU in FS Mode" from the list.
  5. 5.5. Click "Upgrade Driver"
  6. 5.6. Wait a moment for the installation to complete.
6. Click the "Connect" button.
7. Select "FS in DFU mode" from the list.
8. Click the "Write to Device" button.
9. Wait until “Done!” is displayed.
10. The pedal will reboot automatically and then your pedal is now updated!

使い方

1. 「Select Firmware」ボタンを押し、ファームウェアを選択する
2. ペダルの電源を切る
3. USB CケーブルでPCと接続する
4. 両方のペダルのフットスイッチを押しながら、電源を入れる
5. ※Windowsの場合のみ ドライバーのインストール Windowsの場合のみ Macではこの手順は不要です
  1. 5.1. ペダルをブートロードモードでPCに接続(この手順の 4 までを行う)
  2. 5.2. Zadigをダウンロードして開く
  3. 5.3. 「Options」→「List All Devices」をクリック
  4. 5.4. リストから「DFU in FS Mode」を選ぶ
  5. 5.5. 「Upgrade Driver」をクリック
  6. 5.6. インストールが完了するまで少し待つ
6. 「Connect」ボタンを押す
7. 「DFU in FS Mode」を選択し接続する
8. 「Write to Device」ボタンを押し、ファームウェアをアップロードする
9. 「Done!」と表示されるまで待つ
10. ペダルが自動的に再起動したら完了!

Prerequisites

In order to run this demo, you will need:

  • An up-to-date version of Chrome, at least version 61 or newer

  • A USB DFU class device

    For a WebUSB friendly USB DFU bootloader that can be flashed onto inexpensive STM32F103 boards, check out the dapboot project.

    Some recent STM32 parts have on-chip USB DfuSe bootloaders that are supported by this site.
